Цель

Демонстрация приемов программирвоания на языке Visual Prolog:

  • Создание объектно-ориентированного инструментария для консольных приложений *;
  • Создание Windows-приложений на базе GUI пакетов;
  • Создание приложений со строго изолированными логической частью и пользовательским интерфейсом;
  • Создание библиотек (*.lib) на базе VIP.
Примечание:
*Объектно-ориентированная консоль -  учебная шутка!

Функциональность

  • Логика игры не изменилась, три типа пользовательских интерфейсов: Консоль, объектная Консоль, Windows

Построение

Ознакомьтесь предварительно с особенностями имеющихся у вас версий

Откройте и постройте проекты:

  • Polyline11_Console - консольное приложение
  • Polyline11_ObjectConsole объектно-ориентированное консольное приложение
  • Polyline11_ObjWindows - Windows PFC/GUI

Выполните командный файл Programs\BuildHumanInterfaceLib.cmd

Откройте и постройте проекты, использующие библиотеки пользовательских интерфейсов:

  • Polyline11_ConsoleLib;
  • Polyline11_ObjectConsoleLib;
  • Polyline11_ObjWindowsLib

Для построения проектов переменная SpbSolutions должна быть установлена как маршрут к директории, где располагается SpbVipTools

Запуск

  • Язык приложения определяется помещением в командной строке идентификатора языка:
    • ru - русский
    • en - английский
  • При запуске из среды IDE параметр языка помещается в поле формы Project Settings\Run Options\Run Arguments, получаемой из меню Project\Settings...
