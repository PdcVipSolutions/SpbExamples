Tutorial Project. Release 13

The Goal

The demonstration of practices of the Vip-based programming:

  • the ways of the communication of the main applicaion with the parts of the application, which are placed to the DLL (Dynamically Linked Library).
  • the practice of the rising and handling the exceptions.

The Code

The Release 13 contains the set of strongly dependent projects - the project of the main applicaion and projects of three DLLs, where DLL has  the special project to buiuld it.

The set of projects contains:

  • The Project ModelsDLL.vipprj (generates the  DLL), where models of the computer strategies of the game Computer0, Computer1, Computer2 are placed.
  • The Project Computer3DLL.vipprj (generates the DLL), where the model of the strategy Computer3 is placed.
  • The Project HumanInterfaceDLL.vipprj (generates the DLL), where classes realated to the HumanInterface are placed. This DLL is called using the library, which used while the main applicaion building.
  • Project Polyline13_ObjWindows.vipprj - the project of the main executable applicaion.

The Functionality

  • The new strategy model Computer3 appeared
  • The behavior of applications of the Release 13 is the same as the behavior of applications of the Release 11, 12.

Run notes

  • The application corresponding to the project Programs\ObjectWin\Polyline13_ObjWindows.vipprj must be invoked;
  • The language of the application may be defined in the command line by the language indentifier:
    • en - English
    • ru - Russian
  • When running the program from the IDE, the parameter of the language must be placed to the field Project Settings\Run Options\Run Arguments, available from menu Project\Settings...
